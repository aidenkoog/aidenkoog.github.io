## Build / Execute Webpage in VS-CODE

- How To Execute Webpage.
  - First, You have to open index.html in VS-CODE.
  - Open Live Server
    - If there's no Live-Server plugin, please install Live Server in VS-CODE.
    - Short-cut: CMD + L, O
